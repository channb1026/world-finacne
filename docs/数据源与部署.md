# 数据源与部署说明

## 数据源（当前与建议）

| 模块 | 当前 | 建议接入 |
|------|------|----------|
| 新闻 | 地图与热点财经：Yahoo、MSN、BBC、NPR、CNBC、Guardian 等各大门户；A 股资讯：新浪财经、中国新闻网、中金在线、Google 新闻（国内源可经代理或 RSSHub 接入证券时报/东方财富/同花顺等） | 东方财富、同花顺、证券时报、上海证券报、中证网、新华网证券、每日经济新闻、21财经等可经 RSSHub 或官方 RSS 加回 |
| 汇率 | 前端 mock | Exchangerate-api、Fixer、Open Exchange Rates（免费额度） |
| 股市 | 前端 mock | Alpha Vantage、Finnhub、Twelve Data；A 股/港股需持牌源 |
| 直播 | iframe/外链配置 | 仅嵌入官方或授权流，注明来源 |

## 合规注意

- API 密钥全部走后端环境变量，前端不暴露。
- 直播：仅嵌入官方页面或授权流，避免盗播。
- 行情与新闻商用或对外发布前，确认各数据源 ToS；A 股实时行情需用持牌源。

## 部署

- **前端**：静态构建 `frontend/dist` 可部署至 Vercel / Netlify 等；若需请求后端，配置 `VITE_API_BASE` 为生产 API 地址，或通过同域 Nginx 反向代理 `/api`。
- **后端**：Node 进程（如 `node backend/src/index.js`）可部署至单机或云函数；生产环境建议加限流、Redis 缓存与真实新闻源聚合。
